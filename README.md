# PRODIGY_DS_01

## Task Objective
To create a bar chart or histogram that visualizes the distribution of a continuous or categorical variable.

## Dataset
Population dataset containing country population values.

## Tools and Technologies
- Microsoft Excel

## Project Description
In this task, I worked with population data and used the 2024 population values to create a histogram. The dataset was organized and prepared for visualization, and the chart was formatted by selecting an appropriate bin width and adding proper titles and axis labels.

## Result
The histogram clearly shows the distribution of country populations and helps in understanding how population values are spread across different ranges.

## Learning Outcomes
Through this task, I improved my understanding of:
- Data preparation
- Histogram structure
- Bin width adjustment
- Axis labeling
- Data visualization and presentation
